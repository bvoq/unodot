# unodot :: Unopinionated Dotfile Manager 

A good dotfile manager does not get in your way, but still, subtly, pushes you towards best practices, when no alternative makes sense.
Unodot is a tiny set of utilities, in a single file per system, under 1000 LoC, that help you write your own dotfiles.

It is self documenting, run either `unodot.zsh` or `unodot.ps` depending on your system.
